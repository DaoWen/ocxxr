Detailed Description

template<typename T>
class ocxxr::Event< T >

Base class for OCR events.

To create an event, use one of the derived event sub-types (e.g., OnceEvent or StickyEvent) rather than this abstract type. However, when an event is stored in a struct, or passed as an argument, the specific type of the event is often not needed, and this abstract Event type can be used instead.
